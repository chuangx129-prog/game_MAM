# 《妈妈的无影脚》(暂定名)— 游戏设计文档 v0.1

> 也可叫:《今晚别踢我》《女儿的求生之夜》

## 一句话介绍

一个蛋仔风格的**夜间潜行生存小游戏**:女儿和有"睡觉踢人"坏习惯的妈妈同床,她要在整晚躲开妈妈的无影脚、攒够睡眠撑到天亮——但动作太大会吵醒妈妈。

- **平台**:手机/平板浏览器(H5,免下载,横屏)
- **引擎**:Phaser 3(触屏虚拟摇杆 + 单键操作)
- **单局时长**:7~8 分钟(游戏内 23:00 → 6:30)
- **美术方向**:蛋仔派对式 Q 弹圆润小人 + 温馨深夜居家场景(原型期用程序生成的占位图形)

## 核心矛盾(游戏性的根)

> **想赢就得睡觉,但睡觉的地方最危险。**

- 赢的条件是**睡眠值**攒够 → 只有躺在妈妈床上才能涨睡眠
- 床上会随机挨踢 → 要玩"一键闪避"反应小游戏
- 下床去别的房间做任务能拿增益和道具 → 但离床期间睡眠不涨,时间在流逝
- 一切行动都产生**噪音** → 吵醒值满了妈妈醒来直接失败

玩家每分钟都在做决策:**继续苟在床上刷睡眠,还是冒险出门跑图?**

## 三条数值主线

| 数值 | 说明 | 归零/满格后果 |
|---|---|---|
| 😴 睡眠值 0→100 | 只在床上涨,结算评级依据 | <40% 结局是"上课睡着" |
| ❤️ 体力 100→0 | 被踢一脚 -30 | 归零 = 被踢晕,失败 |
| 😠 吵醒值 0→100 | 噪音累积,安静时回落(**乖乖睡着时回落×3**) | 满格 = 妈妈**暴怒追逐**(见下) |

## 暴怒追逐(吵醒值满格触发)

吵醒值满不再直接失败,而是进入全屋追逐战:

- 妈妈暴怒起床追人,**BGM 变恐怖(低音蜂鸣+心跳),整个环境闪红光**
- 妈妈有**红色视野扇形**实时显示,被墙体遮挡;进入视野被锁定,贴身 42px 被抓 = 失败
- 逃脱手段:
  - **跑**(追击速度介于走路和跑步之间,必须跑起来)
  - **躲**进衣柜/沙发后,她看不见;追丢 9 秒后她骂骂咧咧回去睡
  - **逃进爸爸房间**:爸爸把门顶住,妈妈砸门 3 秒打不开,悻悻回房
- 妈妈回去睡后吵醒值重置为 35,夜晚继续——追逐是惊险的"第二次机会",不是终点
- 巡逻时被撞见同样触发暴怒追逐(原来是直接失败)

辅助数值:💧憋尿值(太高睡眠减半,要去卫生间)、⭐星星(货币)。

## 床上玩法:无影脚 QTE(核心循环)

1. 躺下后睡眠开始增长,妈妈每 5~12 秒随机"预警"
2. **红色⚠️ = 真踢**:0.9 秒内按【躲】→ 侧滚闪避,成功连击每 3 次 +1⭐
3. **黄色💤 = 只是翻身**:是假动作,按了白躲(浪费时间+轻微噪音),考验读招
4. 挨踢:-30 体力 + 产生噪音;「替身抱枕」道具可自动挡一次
5. 越接近清晨,妈妈踢腿越频繁(难度曲线)

## 剧情主线:「安眠计划」🌙(道具驱动的任务链)

妈妈踢人是因为**睡不安稳**——女儿可以对症下药,一步步"治好"无影脚。每一步由上一步的道具/剧情解锁,是睡觉刷分之外的**主动玩法路线**:

1. **🧸 小熊**(翻沙发×3 获得):原来是妈妈小时候的宝贝 → 新任务:趁熟睡把小熊塞进妈妈怀里(4秒读条,床边高风险操作)→ 妈妈搂住小熊,**踢腿间隔 ×1.4**。注意:小熊自己抱着有闪避加成,送出去就没了——**取舍**
2. **🥛 床头牛奶**(小熊步骤后解锁微波炉新选项):热好端到床头(读条)→ 妈妈睡得更沉,**噪音影响 ×0.75,踢腿再 ×1.25**
3. **🧺 毯子**(牛奶步骤后,妈妈"踢被子好像很冷"→ 客厅角落篮子里拿毯子)→ 盖上(4.5秒读条)→ **无影脚封印!妈妈不再踢腿、不再起夜,睡眠速度 ×1.5**,达成特殊结局「安眠之夜」

配套机制:**妈妈起夜会踩到没收拾的玩具**(每次起夜最多绊一次)→ 她更烦躁,踢腿间隔 ×0.85——给"捡玩具"支线真实的剧情后果。

## 综合评分(不再只看睡觉!)

结算得分 = 睡眠值 + 任务奖励:安眠计划三步 +8/+8/+10、喂猫 +4、玩具全收 +6、找到小熊 +2。
满分可超 100,≥105 评 **S+「传说」**。躺平流刷睡眠、跑图流做任务,两种玩法都能拿高分。

## 房间与玩法

- **妈妈卧室**:主战场。床(睡觉+QTE)、衣柜(躲藏点)、地上玩具(踩到会响)
- **爸爸卧室(回血安全屋)**:整晚亮灯。进门爸爸就掏冰激凌 🍦(体力回满 + 吵醒值-30 + 90秒"吃饱睡得香"睡眠×1.25,冷却 2 分钟);随时可用⭐向爸爸**兑换道具**;被追逐时逃进来爸爸会**堵门**保你平安
- **卫生间**:憋尿值高了必须来;上完选择"冲水"(+2⭐好孩子奖励,但巨大噪音)或"轻轻盖上"(安全)——风险换收益的小抉择
- **厨房**:微波炉热牛奶(睡眠速度×1.5,"叮"一声有噪音)、冰箱翻猫粮
- **客厅**:沙发后可躲藏;翻沙发缝 3 次找到**小熊🧸**(抱着睡闪避窗口变长);喂猫 🐱(不喂它半夜会叫,叫声涨吵醒值)
- **走廊**:玩具雷区,捡起每个 +1⭐并排雷

## 随机事件:爸爸的精神小伙时刻 🪩

一晚随机 4 次,爸爸突然起来蹦迪 10 秒:专属 BGM(`跳舞.mp3`)+ 爸爸房间夜店彩灯(色相旋转光斑),
噪音持续吵妈妈(吵醒值上涨)。**如果此时妈妈正在暴怒追逐,爸爸房间不再安全**——没人堵门,
妈妈循声一脚踹开房门,你和爸爸一起挨训 → 专属失败结局「🪩 迪斯科惨案」。

## 胜利条件补充

睡眠值攒满 100% **立即胜利**(不用熬到 6:30),结算文案「睡饱收工!」。

## 随机事件:妈妈起夜巡逻

一晚 **4 次**起夜窗口(约 23:55 / 1:20 / 2:50 / 4:30 前后随机浮动;完成安眠计划后不再起夜)。提前有咳嗽预警 + 心跳音效。此时:

- 在床上装睡 = 安全
- 在爸爸房/客厅/厨房 = 安全
- 在她的动线上(卧室/走廊/卫生间)必须**躲进衣柜或沙发后**,被撞见直接失败

## 结局评级

清晨 6:30 结算:睡眠 ≥90% = S「睡美人」/ ≥75% A / ≥55% B / ≥40% C「黑眼圈上学」/ <40% D「上课睡着被点名」。
失败结局:被踢晕 / 追逐战被妈妈逮住——每种都配一句吐槽文案;结算里统计"逃过追击次数"。
